# Learning Log (Django)

This project is built while following the book  
**Python Crash Course (3rd Edition)** by Eric Matthes.

## Purpose

The goal of this project is to:
- Learn Django fundamentals
- Understand project structure and apps
- Practice models, views, templates, and authentication
- Gain experience with Git and GitHub workflow

This is a **guided learning project**, not a production-ready application.

---

## Features

- User authentication (accounts app)
- Create and manage learning topics
- Add entries under each topic
- Django admin integration

---

## Tech Stack

- Python
- Django
- SQLite (local development)
- HTML / CSS
